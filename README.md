# React Query Todo App

A modern Todo application built with Next.js, React Query, and Composer AI integration. This application demonstrates best practices for state management and data fetching in React applications.

## Features

- Todo CRUD operations (Create, Read, Update, Delete)
- React Query for efficient data fetching and caching
- Modern UI with responsive design
- Next.js for server-side rendering and routing
- TypeScript for type safety

## Getting Started

1. Clone the repository:
```bash
git clone cai-composer-react-query-todo
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js
- React Query
- TypeScript
- Tailwind CSS
- Composer AI
